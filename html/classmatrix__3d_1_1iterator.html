Detailed Description

template<typename T>
class matrix_3d< T >::iterator

Class to represent a bidirectional iterator.

Class for representing a bidirectional iterator for the matrix_3d class. The class relies on the iterators of the matrix_2d class, used as main data structure for the matrix_3d class.
