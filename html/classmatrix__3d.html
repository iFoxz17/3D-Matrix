Detailed Description

template<typename T>
class matrix_3d< T >

Class for representing a three-dimensional array.

Class that encapsulates a three-dimensional array. It used a one-dimensional array of two-dimensional matrix (matrix_2d) to simulate the presence of the three-dimensional array.
